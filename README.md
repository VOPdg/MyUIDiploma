## :keyboard: Запуск из терминала

Локальный запуск всех тестов:

```
gradle clean abotests
```

Удаленный запуск всех тестов:

```
gradle clean
abotests
-Dbrowser=${BROWSER}
-DanyText="${ANY_TEXT}"
-Dsize="${SIZE}"
-Dversion="${VERSION}"
-DbaseUrl="${BASEURL}"
```
